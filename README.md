- CRUD User's Job Offers

### Job Search Website | [[app/web-app]](/apps/web-app/)

**Features:**

- Homepage

**Stack:** NextJS, TailwindCSS

**Features:**

- Job Offers
- Company
- User
- Stats
- Auth0
